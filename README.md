# 💻 Programação de Sistemas Embarcados

Bem-vindo! 👋

Este repositório foi criado para armazenar todos os exercícios, atividades e projetos desenvolvidos durante a disciplina de **Programação de Sistemas Embarcados**.

O objetivo é manter uma organização do conteúdo estudado ao longo do semestre, facilitando consultas futuras e acompanhando a evolução do aprendizado.

---

## 📂 Estrutura do repositório

Os exercícios serão organizados em pastas, geralmente seguindo a estrutura abaixo:

```text
📦 Programacao-Sistemas-Embarcados
├── 📁 exercicio-01
│   ├── 📄 README.md
│   ├── 📁 src
│   └── 📁 docs (opcional)
├── 📁 exercicio-02
├── 📁 exercicio-03
└── ...
```

---

## 📚 Conteúdo

Ao longo da disciplina, este repositório poderá incluir exercícios sobre:

* ⚙️ Linguagem C/C++;
* 🔌 GPIO;
* ⏱️ Timers e interrupções;
* 📡 Comunicação UART;
* 🔄 Protocolos I²C e SPI;
* 📈 ADC (Conversão Analógico-Digital);
* 🎚️ PWM;
* 💾 Manipulação de periféricos;
* 🤖 Programação de microcontroladores;
* 🚀 Outros tópicos relacionados a sistemas embarcados.

---

## 🎯 Objetivo

* Organizar todos os exercícios da disciplina;
* Registrar a evolução do aprendizado;
* Servir como material de consulta para estudos futuros;
* Compartilhar soluções e boas práticas de programação.

---

## 🛠️ Ferramentas

Dependendo da atividade, poderão ser utilizadas ferramentas como:

* Linguagem **C/C++**
* **Git** e **GitHub**
* IDEs para desenvolvimento embarcado
* SDKs e bibliotecas específicas do microcontrolador utilizado
